FAQ - Perguntas Frequentes.

O que é HTML

É a linguagem usada para estruturar páginas web.

Para que serve o CSS?

O CSS é usado para estilizar e deixar o site visualmente atraente.

O que é JavaScript?

É uma linguagem que adiciona interatividade ao site.

Como publicar um site?

Usando serviços como o GitHub ou o Netlify

O que é responsividade?

É quando o site de adapta a diferentes tamanho de tela.
